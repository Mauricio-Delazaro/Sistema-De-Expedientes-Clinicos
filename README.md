# Sistema de Expedientes Clínicos

Este proyecto consiste en el diseño de un sistema de gestión de expedientes clínicos orientado a terapeutas y administradores de una clínica de psicología. El sistema tiene como objetivo garantizar la administración segura, estructurada y confidencial de la información clínica de los pacientes.

Los pacientes son registrados previamente en el sistema de agenda de la misma clínica, por lo que este sistema únicamente gestiona expedientes clínicos y sesiones asociadas a dichos pacientes.
